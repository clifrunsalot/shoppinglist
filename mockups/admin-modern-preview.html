Why this works better

The list becomes a scan surface instead of an endless stack of forms. Editing happens in a dedicated space, and bulk deletes happen in an explicit selection mode rather than through repeated per-row actions.

Desktop bulk behavior

Keep the detail panel visible on larger screens, but swap it into a bulk summary when multiple items are selected. Search and filters stay pinned above the list so mass delete can target only the visible subset.

Mobile bulk behavior

Use a pinned bulk-action bar and a full-height confirmation sheet. This avoids mixing destructive selection controls with dense inline editing on a narrow screen.
